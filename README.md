# Morse Code Python Project


## An interactive Morse code project written in Python
This project provides several tools to translate and learn Morse code:
* Translate English to Morse
* Reverse translation from Morse to English
* Quiz (Multiple choice or typing the answer)
* Settings to change the characters used to display Morse code (Default: "-", ".")


## How to run the project
1. Download the repository as a ZIP file
2. Extract ZIP contents into a folder
3. Run "main.py"


## Known issues / Limitations
The program only supports conversion in
* Lower-case English (Converted to lowercase if uppercase was used)
* Numbers
* Spaces
